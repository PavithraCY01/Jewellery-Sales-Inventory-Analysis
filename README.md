# Jewellery-Sales-Inventory-Analysis
📌 # Project Overview

This project focuses on analyzing sales performance and inventory management of a jewellery business using Excel, SQL, and Power BI for data analysis and visualization. It includes:

• Data cleaning and preprocessing
• Handling missing values and data formatting
• Creating calculated columns and measures
• Exploratory data analysis (EDA)
• Inventory analysis using reorder signals
• KPI tracking and dashboard creation
• Business insights generation
• GitHub documentation

🎯 The Goal of this project is to:

• Analyze product-level sales performance
• Identify top-performing and underperforming products
• Evaluate inventory efficiency and stock management
• Understand demand patterns across categories and cities
• Assess the impact of delivery options on sales

🗂 Dataset Description

The dataset includes:
• Product details (category, type, pricing)
• Sales and order data
• Inventory and stock levels
• Reorder signals
• Delivery type (express/standard)
• City-level sales distribution

📈 Key Analysis Performed

• Product-level sales performance analysis
• Category-wise demand analysis
• City-wise sales contribution analysis
• Revenue and order trend analysis
• Inventory status and stock evaluation
• Reorder signal analysis for stock optimization
• Express delivery impact on product demand
• Dashboard creation for KPI monitoring

📊 Key Insights

• A small group of products contributed to the majority of total revenue, indicating sales concentration.
• Low-performing products showed high inventory levels, highlighting overstock issues.
• Fast-moving items faced potential stockout risks due to gaps in replenishment planning.
• Products with express delivery availability had higher sales and customer preference.
• Certain categories and cities consistently drove higher demand and revenue.

📌 Sample KPIs

• Total Revenue
• Order Volume
• Top-Selling Products
• Inventory Status (In Stock / Low Stock)
• Reorder Alert Count
• Category-wise Sales Contribution

✅ Conclusion

This project demonstrates end-to-end data analysis skills using Excel, SQL, and Power BI. It highlights the ability to transform raw data into actionable insights, supporting better decision-making in sales optimization and inventory management—key skills required for data analyst roles.
